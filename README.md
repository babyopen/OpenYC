# Open预测 - 十二生肖AI预测系统

基于深度学习的十二生肖预测系统，采用Transformer + LightGBM混合模型，包含颜色、五行、大小、单双、区间、尾数等多维特征。

## 项目特点

- 🧠 **深度学习模型**：双向LSTM + Transformer架构
- 🔢 **多维特征工程**：108维特征（生肖、颜色、五行、大小、单双、区间、尾数、家禽野兽、连出）
- 🎯 **智能惩罚机制**：基于历史数据的连续相同/不同生肖概率分析
- 📊 **集成学习**：70%深度模型 + 30% LightGBM混合预测
